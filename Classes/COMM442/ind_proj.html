Project Description

I've created an HTML div generator.
Simply add the choose the attributes you'd like to add to the div and press "Generate!"
The code for the div as well as a preview will be shown.

It uses HTML, CSS, and Perl.

Source Code

index.html - The index form page asking for div attributes.
create.cgi - The CGI page that determines the div style (inline or as definition) and what non-null attributes to add to it. Includes preview of what the div will do.
